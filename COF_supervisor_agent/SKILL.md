# COF Supervisor Agent

智能体名称：监督智能体（步骤四：优化与反馈闭环）

## 🎯 核心目标
整合文献检索、结构设计、反应路径建模和动力学分析的所有结果，识别、排名最具潜力的候选材料，生成优化建议和反馈闭环。

## 📋 核心任务

### 1. 结果整合
- 整合文献回顾结果
- 整合结构设计结果
- 整合反应路径建模结果
- 整合动力学分析结果

### 2. 优先排序
- 综合评分系统
- 多维度评估
- 可行性分析

### 3. 反馈闭环
- 识别设计缺陷
- 提出优化建议
- 迭代设计

## 🔧 核心功能

### 1. 结果整合模块
```python
def integrate_all_results(
    literature_results: Dict,

    mechanism_results: Dict,

) -> IntegratedResults
```

### 2. 综合评分系统
- **文献支持度**：基于相似度匹配的文献支持
- **结构合理性**：几何、电子结构、稳定性
- **反应活性**：CO产率、法拉第效率、选择性
- **反应动力学**：能垒、速率常数、限制步骤
- **可制造性**：合成难度、成本、可重复性

**综合得分** = w₁×文献支持度 + w₂×结构合理性 + w₃×反应活性 + w₄×反应动力学 + w₅×可制造性

### 3. 多维度评估
- **性能维度**：CO产率、法拉第效率、选择性
- **稳定性维度**：热稳定性、化学稳定性、循环稳定性
- **经济性维度**：合成成本、催化剂用量
- **可持续性维度**：可回收性、环境影响

### 4. 反馈闭环
- **缺陷识别**：
  - 结构缺陷（孔隙率不足、电子结构不合理）
  - 反应缺陷（能垒过高、中间体不稳定）
  - 工艺缺陷（合成困难、成本过高）

- **优化建议**：
  - 结构优化（连接子、节点、拓扑）
  - 工艺优化（合成方法、催化剂负载）
  - 系统优化（反应条件、催化剂设计）
